Go Microservices API

Bem-vindo à página de demonstração da Go Microservices API, desenvolvida por Gabriel Demetrios Lafis. Esta API é um exemplo de microsserviços de alta performance construídos com Go e o framework Gin.

Funcionalidades Principais

  • Autenticação JWT (a ser implementada)
  • Integração com PostgreSQL (a ser implementada)
  • Endpoints para gerenciamento de usuários
  • Verificação de saúde da API

Como Usar (Exemplos)

Para interagir com a API, você pode usar ferramentas como curl ou Postman. Certifique-se de que a API esteja rodando localmente (instruções no README.md).

Verificar Saúde da API

curl http://localhost:8080/api/health

Listar Usuários

curl http://localhost:8080/api/users

Criar Novo Usuário

curl -X POST -H "Content-Type: application/json" -d 
'{"name": "Novo Usuário", "email": "novo.usuario@example.com"}' 
http://localhost:8080/api/users

Para mais detalhes sobre a implementação e como configurar o projeto, visite o repositório GitHub.
